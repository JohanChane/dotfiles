# LazyVim CheatSheet

-   `uc`: Toggle Conceal Level
-   `fr`: Find Recent

-   `ff`: Find Files (Root Dir)
-   `fF`: Find Files (Cwd)

-   `sg`: Grep Files (Root Dir)
-   `sG`: Grep Files (Cwd)
-   `sb`: Grep Current Buffer

-   `ss`: Goto Symbols (Root Dir)
-   `ss`: Goto Symbols (Cwd)
-   `ct`: Ctags Current Buffer

-   `sw`: Search Word (Root Dir)
-   `sW`: Search Word (Cwd)

-   `cf`: Format

-   `sm`: Jump to Marks

-   `sk`: Key Maps
-   `sh`: Help Pages
-   `sM`: Man Pages
